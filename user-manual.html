How Content Protection Works

Premium course videos are YouTube unlisted links. Anyone with the link can watch. The site prevents non-paying users from ever seeing these links through 3 layers of protection:

Layer 1
Route Middleware

When someone visits /courses/guitar-basics/watch, the middleware checks if they're logged in. If not → redirected to login. They never reach the player page.

Layer 2
Server-Side Purchase Verification

Even if logged in, the server checks the database for a completed purchase for this user + course. No purchase → redirected to the course detail page. YouTube URLs are only loaded if the check passes.

Layer 3
Database Row Level Security

At the database level, RLS policies only return youtube_url values to users who have a completed purchase. Even if the app layer is bypassed, the database refuses to return the data.

How Razorpay Protects Payments

Razorpay handles the money. Here's the exact flow:

1
User clicks "Buy Now" → server creates a Razorpay order
2
Razorpay checkout modal opens → user pays via UPI / Card / Netbanking
3
Razorpay returns a payment ID + cryptographic signature
4
Server verifies the signature using HMAC-SHA256 with your secret key
5
Signature valid → purchase marked completed
6
User can now access all course videos — lifetime access

Why this is secure: Only Razorpay can produce a valid signature. Only your server has the secret key to verify it. No one can fake a payment — the purchase is only marked "completed" after cryptographic verification passes.

How to Upload a Paid Course

The site does not host any video files. All videos live on YouTube as unlisted uploads. You paste URLs into the admin panel.

Step 1: Record & Upload to YouTube

  1. Record your course lessons
  2. Upload each lesson to your YouTube channel
  3. Set each video to "Unlisted" — YouTube Studio → Content → video → Visibility → Unlisted
  4. Copy the URL of each video (e.g., https://www.youtube.com/watch?v=abc123)

Step 2: Create the Course

  1. Go to /admin → Courses → New Course
  2. Fill in: Title, Slug (auto-generated), Description, Price (in Rupees)
  3. Optionally add a Thumbnail URL
  4. Leave Published unchecked while adding videos
  5. Click Create Course

Step 3: Add Videos

  1. From Courses list, click the video icon next to your course
  2. Click Add Video — a form appears with a live YouTube preview
  3. Paste the YouTube URL — the video loads immediately so you can verify it's correct
  4. Fill in title, duration (optional), and check Free Preview if it's a teaser
  5. Repeat for all lessons

Step 4: Publish

  1. Edit the course → check Published → Update Course
  2. The course now appears on the public site with the "Buy Now" button

Step 5: Student Purchase Flow

→
Student visits course page, sees curriculum + price
→
Clicks "Buy Now — ₹999"
→
Razorpay checkout → pays → verified
→
Redirected to course player with all lessons unlocked
→
Can rewatch anytime — lifetime access

Adding More Lessons Later

Upload the new lesson to YouTube (unlisted) → go to Admin → Courses → video icon → Add Video. Students who already purchased see the new lesson immediately.

Replacing a Video

Upload new version to YouTube (unlisted) → delete old entry in Admin → add new one with the new URL.

Managing Free Videos

Free videos are public YouTube videos (not unlisted). Anyone can watch them on your site without logging in.

  1. Go to /admin → Free Videos
  2. Click Add Free Video
  3. Paste the YouTube URL — a live preview appears immediately
  4. Add a title and optional description
  5. Click Add Video — it's published instantly

You can expand any video in the list to see exactly how it looks embedded. Toggle visibility with the eye icon, or delete with the trash icon.

Admin Video Preview

Both the Free Videos and Course Videos admin pages now include:

Live Preview on Add

When you paste a YouTube URL in the add form, the video loads immediately. You can watch it right there to confirm it's the right one before saving.

Inline Preview on Browse

Click the expand arrow on any video in the list to see the full embedded player. Verify what your students will see without leaving the admin panel.

Setup Checklist

  1. Create a Supabase project → copy URL + anon key + service role key
  2. Run supabase-migration.sql in Supabase SQL Editor
  3. Create a Razorpay account → copy test key ID + secret
  4. Update .env.local with all credentials
  5. Run npm run dev → open http://localhost:3000
  6. Register an account → set as admin: 
    UPDATE public.profiles SET role = 'admin' WHERE email = 'your@email.com';
  7. Go to /admin → start adding courses and videos
  8. For production: deploy to Vercel, switch Razorpay to live mode
